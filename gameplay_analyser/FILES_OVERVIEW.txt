INSTALLATION CHECKLIST
----------------------
□ Install Tesseract:     brew install tesseract
□ Install Tkinter:       brew install python-tk@3.13
□ Install Python deps:   pip install -r requirements.txt
□ Verify Tesseract:      tesseract --version
□ Verify Tkinter:        python -c "import tkinter"
□ Test detection:        python debug_status_detector.py [frame.png] --all

USAGE EXAMPLES
--------------
# Debug single frame
python debug_status_detector.py captures/session_X/frames/frame_0050.png --all

# Analyze full session
python game_status_detector.py captures/session_X

# With debug output
python game_status_detector.py captures/session_X --debug

# Custom workers
python game_status_detector.py captures/session_X --workers 8

OUTPUT LOCATIONS
----------------
Analysis results:  captures/session_X/game_status.json
Debug images:      /tmp/status_debug/

RECOMMENDED WORKFLOW
--------------------
1. Read INSTALLATION.md (if first time)
2. Install system dependencies (Tesseract, Tkinter)
3. Install Python packages (pip install -r requirements.txt)
4. Test on sample frame (debug_status_detector.py)
5. Analyze captured session (game_status_detector.py)
6. Review results (game_status.json)
